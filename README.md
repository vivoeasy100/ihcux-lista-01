# 🧾 ihcux-lista-01

## 📌 Descrição

Sistema de pedidos em console desenvolvido em **C# (.NET)** com foco em **usabilidade (UX)**.

---

## 🍔 Cardápio do Sistema

O sistema possui um menu interativo com os seguintes produtos:

* 🍕 Pizza Calabresa
* 🍕 Pizza Mussarela
* 🍔 Hambúrguer
* 🌭 Cachorro-Quente
* 🥤 Refrigerante
* 🍹 Suco Natural
* 🥟 Salgado
* 🍟 Batata Frita
* 💧 Água
* 🍧 Açaí

---

## 🎯 Objetivo

Melhorar a experiência do usuário aplicando heurísticas de Nielsen em um sistema simples.

---

## 🧠 Heurísticas Aplicadas

### ✔ Visibilidade do Status

Exibição de etapas:

* Passo 1 de 3
* Passo 2 de 3
* Passo 3 de 3

---

### ✔ Controle e Liberdade

* Digite **"voltar"** para retornar
* Digite **"cancelar"** para sair

---

### ✔ Tratamento de Erros

* Validação de entrada
* Mensagens claras e orientativas

---

## ⚙️ Funcionalidades

* Seleção de produto via cardápio
* Inserção de quantidade
* Confirmação de pedido
* Navegação entre etapas
* Tratamento de erros

---

## ▶️ Como Executar

```bash
dotnet run
```
